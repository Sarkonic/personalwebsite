Useful PowerShell Scripts

These are some scripts I have used to help with task sequences, automation, or for making things easy.



Password Reset GUI

Simple GUI for me to reference an OU in Active Directory to reset the passwords for those users. This saves time in trying to search through Active Directory several times a day.

Copy Shortcuts to Public Desktop

This script was used in some task sequences for SCCM to copy application shortcuts (Outlook, Excel, etc.) to the public desktop for all users.

Check for BIOS Password and Remove

This script uses DellSMBIOS to check for a BIOS password and attempts to remove it. It needs an existing BIOS password, and at the time we were unsure of what it was and so tried a few possibilities. This also creates a new local admin account.

Crypto Checker: CoinGeckoAPI

This script uses a crypto API to reference prices of crypto currency. This wasn't for work: it was just made for fun and to check crypto prices if I was too lazy to google it.
